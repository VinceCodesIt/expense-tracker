# 💸 Expense Tracker

A simple and efficient web-based expense tracker that helps users manage their income, expenses, and monthly budget.

---

## 🚀 Features

- Add income and expenses
- Categorize transactions (e.g. food, transport, bills)
- View total balance
- Track spending over time
- Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
